## 📄 Pagina's

| Bestand | Pagina | Beschrijving |
|---------|--------|-------------|
| `index.html` | Homepage | Video hero, team sectie, locaties, gallery, Three.js 3D elementen |
| `diensten.html` | Services | Alle diensten met prijzen, categorieën en booking links |
| `booking.html` | Boekingssysteem | 4-stappen boekingsflow: locatie → barber → service → datum/tijd |
| `barber.html` | Barber Profiel | Individuele profielpagina per barber met bio en portfolio |
| `about.html` | Over Ons | Verhaal van Fresh Fades, milestones, 3D elementen |
| `producten.html` | Producten | Producten overzicht en shop |
| `thanks.html` | Bedankt | Bevestigingspagina na boeking |
| `404.html` | 404 | Custom error pagina |

---

## 🛠 Tech Stack

- **HTML5** — Semantische structuur
- **CSS3** — Custom properties, flexbox, grid, animaties
- **Vanilla JavaScript** — Geen frameworks, puur JS
- **Three.js** — 3D elementen (barberstoel, neon sign)
- **Google Fonts** — Playfair Display, DM Sans, JetBrains Mono

### Geplande integraties
- **Supabase** — Database voor boekingen en admin
- **Brevo** — Transactionele e-mails (bevestigingen, herinneringen)
- **Netlify** — Hosting + serverless functions

## 🚀 Lokaal bekijken

Open simpelweg `index.html` in je browser, of gebruik een lokale server:

```bash
# Met Python
python3 -m http.server 8000

# Met Node.js
npx serve .
```

Ga dan naar `http://localhost:8000`
